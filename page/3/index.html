E. Maximum Subsequence

E. Maximum Subsequence

You are given an array a consisting of n integers, and additionally an integer m. You have to choose some sequence of indices b_1, _b_2, …, _b__k (1 ≤ b_1 < _b_2 < … < _b__k ≤ n) in such a way that the value of is maximized. Chosen sequence can be empty. Print the maximum possible value of .

Input

The first line contains two integers n and m (1 ≤ n ≤ 35, 1 ≤ m ≤ 109). The second line contains n integers a_1, _a_2, …, _a__n (1 ≤ a__i ≤ 109).

Output

Print the maximum possible value of .

Examples

Input

4 4
5 2 4 1

Output

3

Input

3 20
199 41 299

Output

19

Note

In the first example you can choose a sequence b = {1, 2}, so the sum is equal to 7 (and that’s 3 after taking it modulo 4). In the second example you can choose a sequence b = {3}.

    看的BledDest的题解

Let’s consider the naive solution in O(2_n_) or O(2_n_·n). Iterate over all subsets of original set, calculate sums and take maximum of them modulo m. Now we can use meet-in-the-middle technique to optimize it to . Preprocess the first elements naively and push sums modulo m to some array. After this process the second half with following algorithm. Take sum of the set and find the greatest total sum of current and some sum in the array. As any sum of two numbers less than m can go no greater than 2_m_, we can consider just two values: the greatest number in array and the greatest number less than m - currentSum in the array. This can be found by binary search over sorted array. Overall complexity: .

朱神的序列

预处理 left_max[i] 表示a[1…i]中最大的数值，right_min[i] 表示 a[i…n]中最小的数值，然后扫一遍更新答案，ans=max(ans, left_max[i-1]-right_min[i]);

#include
using namespace std;
#define maxn 10009
#define inf 1000000000
int a[maxn], left_max[maxn];
int min(int a, int b){
return a<b?a:b;
}
int max(int a, int b){
return a>b?a:b;
}
void solve(){
int n, right_min=inf, ans;
scanf(“%d”, &n);
left_max[0]=-inf;
for(int i=1; i<=n; ++i){
scanf(“%d”, &a[i]);
left_max[i]=max(left_max[i-1], a[i]);
}
ans=-inf;
for(int i=n; i>1; –i){
right_min=min(right_min, a[i]);
ans=max(ans, left_max[i-1]-right_min);
}
printf(“%d\n”, ans);
}
int main(){
int T;
scanf(“%d”, &T);
while(T–){
solve();
}
return 0;
}
